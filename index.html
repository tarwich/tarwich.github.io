Profile

The main thing people ask me is what language do you program in. Strange question, as most good programmers can program in nearly any language. Having started programming when I was 8, this is certainly the case for me.

However, to answer succinctly I mainly enjoy C++, PHP, and Java.

  • C++

    I use C++ when I'm writing command line utilities that need to do things that regular scripts cannot. I write about 6 command line program per year.

  • PHP

    PHP is mainly useful as a web technology. I have written a PHP framework for most of the corporate web applications that I write, because it is easy for me to train people to maintain it. However, you can use PHP for a tool command language as well. I write about 3 php scripts per week.

  • Java

    Java has a lot of trouble doing operating system specific tasks, so it's great for writing a cross-platform application. I use Java for command line utilities, web servers, and desktop GUI applications. I write about 1 Java program per month.

  • TCL

    I use TCL / Expect to write anything that has to be done with CLI, but takes up more than one line. Bash / Zsh scripts usually can do the trick, but many times TCL ends up creating a much more intelligent product. I write about 1 TCL script per week.
